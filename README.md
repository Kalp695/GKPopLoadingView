GKPopLoadingView
================

A custom loading view which is blocking the display,animations
are build with Facebook's POP


Install
================
`pod 'GKPopLoadingView'`

Author
================
Georg Kitz, you can find me on [Twitter](http://twitter.com/gekitz)
